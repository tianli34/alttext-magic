# 规划技能

## 适用场景
- 涉及多个文件的修改
- 会影响功能逻辑、数据、权限认证、计费或生产流程的改动
- 需要系统性排查定位的调试工作

## 规划原则
- 步骤应**足够小**（每步 2–10 分钟）。
- 每一步都必须包含**验证环节**。
- 优先**增量交付**（避免一次性大规模修改）。
- 提前明确**回滚方案**与**风险控制措施**。
