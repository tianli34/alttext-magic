
查看数据
docker exec -it my-postgres18 psql -U myuser -d mydatabase

| 你想做的    | 正确命令                         | 说明             |
| ------- | ---------------------------- | -------------- |
| 查看所有数据库 | `\l`                         | 小写 L           |
| 查看所有表   | `\dt`                        | display tables |
| 切换数据库   | `\c 数据库名`                    | connect        |
| 查看当前数据库 | `SELECT current_database();` | SQL 语句         |
| 退出 psql | `\q`                         | quit           |


mydatabase=# \d shops
          -- 看结构
mydatabase=# SELECT * FROM shops LIMIT 5;
  -- 看数据
mydatabase=# \dt+ shops
