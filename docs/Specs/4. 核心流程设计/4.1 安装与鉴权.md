
1. 商家安装 App，完成 OAuth。
2. 存储：
   - `shop_domain`
   - `offline access token`（加密）
   - 安装时间
3. 初始化店铺默认数据：
   - `plan = FREE`
   - `scan_scope_flags = { PRODUCT_MEDIA, FILES, COLLECTION_IMAGE, ARTICLE_IMAGE }`
   - `last_published_scope_flags = {}`
   - 创建安装欢迎额度 bucket：50 张
   - 创建当月 Free 月配额 bucket：25 张  
     - `bucket_type = FREE_MONTHLY_INCLUDED`
     - `source_type = FREE_MONTHLY_GRANT`
     - `cycle_key = FREE:YYYY-MM`
4. 注册 Webhooks：
   - `APP_UNINSTALLED`
   - GDPR 三类
   - `BULK_OPERATIONS_FINISH`
   - `products/create`
   - `products/update`
   - `collections/create`
   - `collections/update`

> Webhook 可以全量注册，但是否真正执行增量刷新，由 worker 在消费时按当前计划、当前 scope、debounce 与图片指纹做 gate。
