
结构化日志建议字段：

- `shop_domain`
- `batch_id`
- `job_item_id`
- `alt_plane`
- `write_target_id`
- `group_type`
- `model_used`
- `context_mode`
- `duration_ms`
- `error_code`
- `error_message`

重点指标：

**扫描**

- 每类 task 行数
- attempt 次数
- URL 下载失败次数
- derive 产出 `scan_result_target / scan_result_usage` 数
- publish 耗时
- 本次 publish 替换的资源切片数
- 本次 publish 保留旧切片的失败资源类型数
- parentId 无法关联次数
- `position_index` 缺失或异常次数
- worker 内存峰值

**增量**

- `webhook_event` 延迟消费次数
- `COALESCED` 数量
- `SKIPPED_PLAN` 数量
- `SKIPPED_SCOPE` 数量
- `SKIPPED_NO_IMAGE_CHANGE` 数量
- 产品增量 usage sweep 数量
- 增量重建 `candidate_group_projection` 数量

**展示投影**

- `candidate_group_projection` 当前总量
- 同一 candidate 同时出现在多个 group 的数量
- `primary_usage` 缺失数量
- `additional_usage_count > 0` 的共享候选数量

**装饰性**

- 标记数量
- 取消标记数量
- decorative 占比

**生成**

- 生成前真值复核发现“已非空”数量
- `SKIPPED_ALREADY_FILLED` 数量
- `SHARED_NEUTRAL` 生成数量
- `RESOURCE_SPECIFIC` 生成数量

**计费**

- `FREE_MONTHLY_INCLUDED` 发放数
- `MONTHLY_INCLUDED` 发放数
- `ANNUAL_INCLUDED` 发放数
- 欢迎额度发放数
- 超额包购买成功率
- reservation 创建 / 释放 / 过期数量
- 生成前因额度不足被阻断次数
- 周期 bucket 重复发放冲突次数

**互斥**

- 各类 409 次数
- 锁超时回收次数

**scope**

- `/api/settings/scope` 调用次数
- scope 开关后立即生效的 `SKIPPED_SCOPE` 数量
- scope 重新勾回后恢复显示的 group 数量

**失败重试**

- `GENERATION_FAILED_RETRYABLE` 当前存量
- `WRITEBACK_FAILED_RETRYABLE` 当前存量
- 失败后重试成功率
