已完成 `POST /api/settings/scope` API 的实现和测试。

### 实现文件

**[`api.settings.scope.tsx`](app/routes/api.settings.scope.tsx:1)** — 路由 action 函数，完整流程：
1. **方法检查** — 非 POST 返回 405
2. **鉴权** — `authenticate.admin(request)` 校验 Shopify 登录态，获取 shop 上下文
3. **Shop 查找** — 通过 `shopDomain` 查内部 `shopId`，不存在返回 404
4. **Body 解析** — 解析 JSON，失败返回 400
5. **Flags 提取** — 检查 `flags` 字段存在性，缺失返回 400
6. **服务调用** — 调用 [`updateScanScopeFlags`](server/modules/shop/scope.service.ts:135)，内部 Zod 校验非法 flag 抛 `ZodError` → 返回 400 + issues 详情
7. **保证不变** — [`updateScanScopeFlags`](server/modules/shop/scope.service.ts:144) 仅更新 `scan_scope_flags` 和 `scan_scope_updated_at`，绝不修改 `last_published_scope_flags`
