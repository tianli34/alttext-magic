项目简介
Shopify Admin Embedded App，自动生成图片 Alt Text。

Stack:
- Shopify embedded app
- React Router
- TypeScript
- Prisma + PostgreSQL
- Redis + BullMQ
- Polaris Web Components + App Bridge
