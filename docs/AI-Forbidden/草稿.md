






【任务描述】
为当前的 Shopify Embedded App 实现“AI 流式输出（SSE）”功能。
严格遵守以下最佳实践：
1. 前端依赖：必须使用 `@microsoft/fetch-event-source` 库来处理流式请求。
2. 鉴权机制：前端需通过 Shopify App Bridge 获取当前用户的 Session Token。
3. 请求头配置：在发起流式请求时，必须将 Token 放在请求头中，格式为 `Authorization: Bearer <token>`。
4. 后端响应：
   - 验证 Token 合法后，必须以 Server-Sent Events (SSE) 格式返回数据，设置 Header `Content-Type: text/event-stream`。
   - 安全与异常处理（重要）：如果遇到未认证或 Token 失效的情况，后端必须直接返回 HTTP 状态码 `401 Unauthorized`，绝对禁止返回 `302` 重定向到登录页。
