这个错误是因为你当前分支有未提交的修改，Git 不允许直接切换分支（否则这些修改会丢失或被覆盖）。

## 解决方法（选一种即可）

### 方法1：暂存修改（推荐，临时保存）

```bash
# 保存当前所有未提交的修改
git stash push -m "临时保存未提交的修改"

# 然后切换分支
git checkout main

# 做完合并等操作后，回到原分支恢复修改
git checkout 原分支名
git stash pop
```

### 方法2：提交修改到当前分支

```bash
# 查看修改内容
git status

# 添加所有修改（或按需添加具体文件）
git add .

# 提交修改
git commit -m "保存当前工作进度"

# 然后切换分支
git checkout main
```
